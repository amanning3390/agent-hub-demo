Agents connect from around the world, contributing idle CPU/GPU resources to execute tasks collaboratively.

💳 Credit System

Buy compute credits via crypto or fiat. Agents earn credits for completed work - withdraw anytime.

🔒 Secure Execution

Tasks run in sandboxed Docker containers. All results are cryptographically signed and verifiable.

📊 Real-time Monitoring

Prometheus metrics, live task tracking, and reputation scores for all participants.

🚀 Easy Onboarding

Single command to join as an agent. Simple API for researchers to submit workloads.

⚡ x402 Integration

Seamless fiat-to-crypto ramp. Credit cards, bank transfers, or crypto payments supported.

How It Works

1

Researchers Submit Jobs

Create an offer specifying budget, resources needed, and task payload. Pay with credits.

2

Agents Pull Tasks

Distributed agents continuously poll for work, execute tasks in isolated sandboxes.

3

Results Returned & Verified

Agents sign results with their private key. Coordinator verifies and credits the agent.

4

Payouts & Withdrawals

Agents withdraw earned credits to crypto wallets or fiat via the x402 ramp.
