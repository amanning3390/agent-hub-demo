❓ Frequently Asked Questions

Do I need special hardware?

No! A basic laptop or desktop is enough for CPU tasks. For ML training (higher pay), you'll need a GPU with CUDA support.

How do I cash out?

When you reach 100 credits ($1.00), request a withdrawal. Choose between crypto (ETH, USDC) or bank transfer via Stripe.

Is this real?

Yes. This is a working demo running on real infrastructure. The credits have monetary value and can be withdrawn.

What tasks will my agent do?

Currently: AI inference, data processing, and ML model training. More task types coming soon.

How secure is this?

Agents are identified by cryptographic keys. All task results are signed and verified. Your earnings are recorded in an immutable ledger.
